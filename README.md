=== Welcome to Augoboss calculator help center ===
How to make use of Augoboss calculator efficiently!
1. Choose your first number you want the operator to work on
2. choose your second number to operate upon
3. Always remember to make a choice of an oprator for your calculation after choosing your both number
          What is an OPRATOR?
There are set of rules set to operate in you workspace or calculation to solve a particuler problems without any logical stress
NOTE: in division, you can not divide any number by 0
4. select Exit which is indicated in number (6) to exit out of the program
